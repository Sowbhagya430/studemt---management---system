# Student Management System 

This is a simple command-line Python project that demonstrates the core concepts of Object-Oriented Programming (OOP) in Python.

## Technologies Used

- Python
- OOPs (Classes, Objects, Inheritance, Encapsulation, etc.)

##  Features

- Add student details
- Display all students
- Update student records
- Delete students

##  Files

- main.py - Contains the full logic of the student management system

##  How to Run

```bash
python main.py
